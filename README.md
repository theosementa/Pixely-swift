# Pixely
Pixely is an iOS app for managing photos, creating albums and sub-albums to organize your photos, as well as filters for finding photos and a memories system similar to Snapchat.

<br>

## Features
- [X] Photo/video collection
- [X] Create album
- [X] Create sub-album
- [X] Photo/Video informations
- [ ] Filters
- [ ] IA Classification (Apple Foundation Models?)
- [ ] Memories system
- [ ] Preferences
- [ ] Localization
- [ ] Tests
- [ ] CI/CD

<br>

## Tech
This app is built to illustrate how coding best apps with Swift and SwiftUI with the better pratices and a modular architecture.

<br>

### Best practices
- Swiftlint (Code linter)
- Modular architecture
